# Bear Lake Camp CMS Login Guide

## First-Time Setup

1.  **Accept the repository invitation**
    -   Check your email for an invitation from GitHub to access `sparkst/bearlakecamp`
    -   Click the link and accept the invitation
2.  **Go to the CMS**
    -   Visit: [prelaunch.bearlakecamp.com/keystatic](https://prelaunch.bearlakecamp.com/keystatic)
3.  **Sign in with GitHub**
    -   Click the **"Sign in with GitHub"** button
    -   If prompted, authorize the **bearlakecamp-cms** app
    -   You only need to authorize once
4.  **You're in!**
    -   You should now see the Keystatic dashboard
    -   Click on **Pages** to edit website content

------------------------------------------------------------------------

## Returning Users

Just visit [prelaunch.bearlakecamp.com/keystatic](https://prelaunch.bearlakecamp.com/keystatic) - you'll be automatically logged in.

------------------------------------------------------------------------

## Troubleshooting

**"404" error when signing in?** - Make sure you accepted the GitHub repository invitation

**"Authorization failed"?** - Try signing out of GitHub and back in - Contact Travis for help

**Can't find the invitation email?** - Check your spam folder - Ask Travis to resend it
