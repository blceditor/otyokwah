# Bear Lake Camp Design Mockup

**Hosted at:** https://prelaunch.bearlakecamp.com/mockup/

This directory contains the visual homepage mockup demonstrating the new "nature-authentic" design direction for Bear Lake Camp.

## What's Included

- **index.html** - Full homepage mockup with all components
- **styles.css** - Complete design system (colors, typography, components)
- **README.md** - This file

## Key Features Demonstrated

1. **Trust Bar** - Sticky on mobile with social proof (500+ families, 4.9/5 rating, ACA accredited)
2. **Hero Section** - Video placeholder with handwritten tagline "Where Faith Grows Wild"
3. **Mission Section** - Background photo with overlay text
4. **Program Cards** - Jr. High & High School with hover effects
5. **Gallery** - 6 existing camp photos with responsive grid
6. **Instagram Feed** - Placeholder for #FaithGrowsWild integration
7. **Mobile Sticky CTA** - Appears after scrolling (mobile only)
8. **Footer** - Quick links, social media, contact

## Design System

**Color Palette:**
- Primary: Muted Lake Blue (#4A7A9E)
- Secondary: Deep Forest Green (#2F4F3D)
- Accent: Clay (#A07856)
- Neutrals: Cream (#F5F0E8), Sand (#D4C5B0), Bark (#5A4A3A)

**Typography:**
- System fonts for body/headings (performance)
- Caveat handwritten for taglines (warmth)
- 18px minimum body text (mobile readability)

**Accessibility:**
- WCAG AA compliant (8:1+ contrast ratios)
- 48px × 48px tap targets (thumb-friendly)
- Skip link, ARIA labels, focus states
- Reduced motion support
